## 프로젝트 개요
![logo image](../src/front-end/src/assets/logo-bg.png)

**"온갖"** 은 OTT 서비스 부가기능을 통합하여 작품 통합검색과 계정 공동구독 매칭 기능을 제공하는 **"OTT 모아보기"** 웹 서비스입니다.

>**OTT 모아보기**  
1.여러 OTT에 흩어져있는 작품들을 '모아보기'   
2.OTT 공동구독 인원을 '모아보기'
